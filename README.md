# TXT Editor

TXT Editor is a simple Python-based text editor project created to work with plain text files through a lightweight interface.

It is a good educational project for learning file handling, GUI logic, and desktop application structure.

## Features
- Open text files
- Edit content
- Save changes
- Lightweight and beginner-friendly structure

## Requirements
- Python 3.10 or newer recommended

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python main.py
```
Or execute txt-editor.exe without needing python or libraries

## Educational value
This project is useful for practicing:
- File I/O
- Basic desktop interfaces
- Application structure
- Event-driven programming

## Possible improvements
- Keyboard shortcuts
- Dark mode
- Search and replace
- Tabs for multiple files
- Unsaved changes warning
